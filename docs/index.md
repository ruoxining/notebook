## 页表 Page Table
在网上找笔记/资源的时候，我的最大感受之一是虽然资源很多很多，但是常常不知道某个资源有多重要，无法评估里面涵盖了多少东西从而不知道要不要点开，或者无法估计在期末有限的时间里应该花多少时间复习某个资源。我正在努力在这里建一个尚能读的资源的 pagetable，以防不幸的游客从侧边栏游走进本图书馆的垃圾堆漩涡🌀。如果我目前的笔记在这个维度上做得不好，请联系我或者向我提 issue 催我改 谢谢！
